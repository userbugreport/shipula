---
Title: scale
---

```shell
Usage:  scale [options] <packageDirectory> [stackName]

Set scaling limits on your App Stack.

Options:
  --number <number>  Number of copies to run -- horizontal scale
  --cpu <number>     Number of CPUs per copy
  --memory <number>  Number of GB of memory, limited by number of CPUs
  -h, --help         display help for command

```
