---
Title: backup
---

```shell
Usage:  backup [options] [command]

Manage backups of your App/Stack shared file system.

Options:
  -h, --help                                           display help for command

Commands:
  take [packageDirectory] [stackName]                  Creates a backup copy of your App/Stack file system, and provides a link where you can download it.
  restore [packageDirectory] [stackName] [backupFrom]  Restore a backup to your App/Stack file system
  help [command]                                       display help for command

```
